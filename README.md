## 🎯 Overview
This project is an **Investment Banking (IB) Analytics tool** designed to evaluate historical Mergers & Acquisitions (M&A) within the technology sector. It bridges the gap between raw data and strategic insight by analyzing how acquisitions transform into **"Derived Products"** and utilizing Machine Learning to estimate future deal valuations.

Built with **Streamlit** and optimized for **MacBook M4 Performance**, this dashboard provides a forensic look at market consolidation from 1987 to 2026.

---

## 🚀 Key Features

* **Accumulation Velocity:** Track how tech giants like Google, Microsoft, and Apple aggressively scale their portfolios over time.
* **Synergy Mapping:** Identify target industry segments (e.g., Cloud, AI, Social Media) that drive corporate strategy.
* **Product Evolution Matrix:** A deep dive into "Derived Products"—showing exactly which acquired companies birthed core products like *Android*, *Azure*, or *Siri*.
* **AI Deal Estimator:** A predictive lab using a simulated **Random Forest Regressor** to forecast potential acquisition costs in the 2026-2030 window.
